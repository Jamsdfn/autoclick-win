# Windows桌面连点工具

一个简单易用的Windows桌面自动连点工具，使用Python编写。

## 功能特点

- 🖱️ 自动鼠标连点
- ⚙️ 可调节点击间隔
- 🔢 可设置点击次数
- ⌨️ 支持热键控制（F8）
- 📊 实时显示点击状态
- 🎨 简洁友好的GUI界面

## 安装步骤

### 1. 安装Python

确保你的系统已安装Python 3.7或更高版本。你可以从[Python官网](https://www.python.org/downloads/)下载安装。

### 2. 安装依赖

打开命令提示符（CMD），进入项目目录，然后执行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

或者直接安装PyAutoGUI：

```bash
pip install pyautogui
```

## 使用方法

### 1. 运行程序

在项目目录中执行以下命令：

```bash
python autoclick.py
```

### 2. 设置参数

- **点击间隔**：设置两次点击之间的时间间隔（秒）
- **点击次数**：设置总共要点击的次数
- **开始/停止热键**：默认使用F8键控制连点的开始和停止

### 3. 开始连点

- 点击"开始连点"按钮，或者按下F8键开始连点
- 点击"停止连点"按钮，或者再次按下F8键停止连点

## 注意事项

1. 请确保在使用连点工具时，你的鼠标指针位于正确的位置
2. 连点过程中，你可以移动鼠标，但连点会继续在当前位置进行
3. 如果需要在游戏或某些应用中使用，请确保应用程序允许自动连点
4. 建议不要设置太短的点击间隔，以免对系统造成过大负担
